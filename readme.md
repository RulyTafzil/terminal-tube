Very WIP terminal app for interacting with YouTube livestream chat.

## Rust version

### Setup

- **Enable API**: Enable **YouTube Data API v3** in Google Cloud.
- **OAuth client**: Create an **OAuth 2.0 Client ID** of type **Desktop app**.
- **Download secrets**: Save the downloaded JSON as `client_secrets.json` in the repo root.

### Run

```bash
cargo run -- <VIDEO_ID_OR_URL>
```

### Notes

- **Token storage**: A token is cached in `yt_token.json`. If you already have a token file generated by the Python version, the Rust app will read it too.
- **Quit**: `Ctrl+C`
